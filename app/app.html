Vigilance index (transparent)

Goldilocks window: too fast yields low vigilance, too slow yields low vigilance. Anchor is set on first hover or focus, so Δ measures deliberation time, not queue delay.
Δ = t_decision - t_anchor
β = w_features·n_features + w_edits·n_edits + w_focus·focus_seconds
s_fast = σ(k · (w_latency·Δ - (τmin + β)))
s_slow = σ(k · ((τmax + β) - w_latency·Δ))
V = s_fast · s_slow
Each row tooltip prints Δ, β, z_fast, z_slow, s_fast, s_slow.
